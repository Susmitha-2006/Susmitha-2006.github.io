HTML and CSS

Have you ever wondered how websites actually work? Like, what makes them look so cool and work so smoothly?

Well,It's not magic at all! It's all thanks to two special things: HTML and CSS.

HTML is like the skeleton of a webpage, giving it structure and content elements such as headings, paragraphs, images, and links. while CSS is the artist, adding colors, styles, and all the pretty stuff So, if you've ever been curious about how it all comes together, you're in the right place!

We're here to guide you through the basics of HTML and CSS, so you can understand how the web works and even you can create your own cool sites someday!

Hyper Text Markup Language

HyperText Markup Language, is the standard language used to create and design web pages. It provides the structure and content of a webpage by using a markup system composed of various tags and attributes.

Some key points about HTML:

  1. Markup Language:

    HTML,which stands for hyper text markup language is not a programming language. It is a markup language which consists of a set of markup tags that define the structure and content of a webpage.

  2. Elements:

    HTML documents are made up of HTML elements, which are enclosed by tags. Elements can include headings, paragraphs, images, links, lists, forms, and more.

    3. It provides the structure and content of a web page by using a system of tags and attributes.

  3. Tags:

    HTML tags are used to mark up elements within a document. Tags are enclosed in angle brackets < >, and most tags come in pairs: an opening tag and a closing tag

Cascading Style Sheets

CSS is a language used to style web pages. CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML or XML.

It controls the layout, formatting, and appearance of elements on a web page, such as fonts, colors, spacing, and positioning.

CSS allows web developers to separate the structure of a document (HTML) from its presentation, making it easier to maintain and update the styling of a website across multiple pages.

CSS works by selecting HTML elements and applying styling rules to them. These styling rules consist of properties and values,

where properties define the visual characteristics of elements (e.g., color, size, margin), and values specify how those properties should be styled (e.g., blue, 12px, 20px).

It helps make websites look good by controlling things like colors, fonts, and layout

Selectors

Selectors are patterns used to select the elements you want to style.

They can target elements by their tag name, class, ID, attributes, or relationship to other elements.

Examples include element selectors (e.g., p for paragraphs), class selectors (e.g., .class-name), and ID selectors

Properties and values

CSS rules consist of one or more declarations, each containing a property and a value.

Properties define the visual aspects to be changed (e.g., color, font-size, margin)

Values specify how the properties should be styled (e.g., red, 16px, 20px)

Types of styling in CSS

CSS can be applied in three ways:

Inline styling : CSS is applied directly to HTML elements using the style attribute.

Internal styling : CSS is defined within the style element in the head section of an HTML document. selector { property:value}

External styling : CSS is written in separate CSS files and linked to HTML documents using the link element.
